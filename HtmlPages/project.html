Project 1 - WeatherWebApp


This is basically a simple weather forecasting website which helps to see the weather forecast of a particular day as well as the weather forecast for a week. In this project I have used Application Programming Interface (API) that is from OpenWeatherMap to fetch the live weather forecast.

Project 2 - Bank System


In this project I have created a simple bank system in which we can check balance, withdraw and also can deposit the cash. Initially the user have to deposit a minimum amount of Rs 100 to proceed for the steps of checking balance , withdraw or to deposit the money. I have used the swing library for creating the Graphic User Interface (GUI).
